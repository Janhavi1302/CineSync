Voice Assignment Logic

Step 1 — Gender Detection: Each speech segment is isolated using STT timestamps. Autocorrelation pitch analysis runs on the voice only. Median F0 > 165 Hz → female, ≤ 165 Hz → male.


Step 2 — Speaker Tracking: Segments within 1.5s = same speaker. Gap > 1.5s = new speaker. IDs: speaker_0, speaker_1, etc.


Step 3 — Voice Locking: Once a gender is assigned, it is permanently locked. speaker_0: male → MadhurNeural never changes.

🚀 Quick Start

# 1. Install Node dependencies
npm install

# 2. Install Python dependencies
cd ai-engine
pip install torch --index-url https://download.pytorch.org/whl/cu121
pip install -r requirements.txt

# 3. Start everything
cd ..
.\start.ps1

# Or manually:
# Terminal 1: cd ai-engine && python main.py
# Terminal 2: npm run dev
